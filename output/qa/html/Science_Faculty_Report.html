How to read this
Percentages show current performance. Bracketed counts show how many items were in scope.
Comparison text shows the gap to university, and the badge shows movement since the previous applicable return.
Not applicable means 0 items were in scope for that period, so it is excluded from change calculations.
! marks a source percentage outside 0-100 and should be reviewed with the submitting department.
