How to read this
Percentages show current performance. Bracketed counts show how many items were in scope.
Not applicable means 0 items were in scope for that period, so it is excluded from trend interpretation.
Trend charts show reporting periods in date order and scale to the values shown.
! marks a current or historic source percentage outside 0-100 and should be reviewed with the submitting department.
